# Website CODING CLUB, SATI

Welcome to the official repository for the **CODING CLUB, SATI Website**! This platform serves as the central hub for our college's developer community, featuring upcoming events, coding challenges, project showcases, and a member directory.

Built by students, for students. 💻✨

---

## 🛠️ Tech Stack

List the technologies used to build this website:

*   **Frontend:** HTML5, CSS3, JavaScript (or React.js / Next.js / Vue.js)
*   **Styling:** Tailwind CSS / Bootstrap
*   **Backend:** Node.js / Express.js
*   **Database:** MongoDB / PostgreSQL
*   **Hosting/Deployment:** Vercel

---

## ✨ Features

*   **Event Tracker:** Stay updated with upcoming hackathons, coding contests, and workshops.
*   **Leaderboard:** Displays rankings from our weekly club coding challenges.
*   **Project Showcase:** A space for club members to display their open-source projects.
*   **Resources:** Handpicked roadmaps and study materials for DSA, Web Dev, and AI/ML.
*   **Admin Panel:** Easy-to-use dashboard for club leads to post announcements and manage events.

## 🚀 Getting Started

Follow these steps to set up the project locally on your machine.

### Prerequisites

Make sure you have the following installed:
*   [Node.js](https://nodejs.org/) (v18.x or higher recommended)
*   [Git](https://git-scm.com/)
*   [MongoDB](https://www.mongodb.com/) (if applicable)

### Installation & Setup

1. **Clone the repository:**
```bash
   git clone https://github.com/Kola0D246/Coding-Club-SATI-Website.git
   cd Coding-Club-SATI-Website
```

2. **Install dependencies:**

```bash
   npm install
```

3. **Set up Environment Variables:**
Create a .env file in the root directory and add your credentials (use .env.example as a reference)

```.env
   PORT=5000
   MONGO_URI=your_mongodb_connection_string
   JWT_SECRET=your_secret_key
```

4. **Run the development server:**

```bash
   npm run dev
   # or
   npm start
```

Open http://localhost:3000 in your browser to see the result.
